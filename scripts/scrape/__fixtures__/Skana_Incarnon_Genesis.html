Skana Incarnon Genesis is an item that is used to add Incarnon upgrades to the  Skana,  Prisma Skana or  Skana Prime.

Acquisition

[edit | edit source]
  • Upon completion of The Duviri Paradox and Angels of the Zariman, and unlocking The Steel Path, players can earn  Skana Incarnon Genesis from The Circuit's Steel Path to place on their Skana with the help of Cavalero in the Chrysalith. This Incarnon Genesis form grants it evolutions to a transformation and newfound strength.
  • Purchased from Cavalero for 120 Platinum 120, complete with the resources used in its installation. This purchase can only be made once.

Incarnon Genesis

[edit | edit source]
  • Installing the Skana Incarnon Genesis requires 20  Pathos Clamps, 70  Dracroot, and 80  Connla Sprouts.
    • When installed, the first Evolution tier is unlocked. Each challenge additionally grants 5,000 affinity to the weapon, affected by Affinity Boosters.

Overview

[edit | edit source]
  • Incarnon Form is charged by building Melee Combo to 6x, then using a Heavy Attack. Once activated, the Incarnon form lasts for 180 seconds.
    • Incarnon Form persists through all activities, such as holstering, bleedout, or being disarmed. However, Incarnon Form is removed upon death.

Patch History

[edit | edit source]

Update 39.0 (2025-06-25)

  • Incarnon Adapters available in the current Steel Path Circuit rotation are now available for Platinum Purchase in Cavalero’s store in the Chrysalith.
    • Purchasing these Incarnons for Platinum will reward players with both the Adapter and the required resources to install it.

Hotfix 37.0.8 (2024-10-16)

  • Fixed typo in Skana’s Incarnon Evolution I description.

Update 33.5 (2023-06-21)

  • Fixed Melee Incarnon weapons (original and Genesis) missing their Incarnon Form stats in the Arsenal.
  • Fixed the Evolution IV description of the Prisma Skana’s Incarnon Genesis stating 30% Status Chance increase instead of the intended 20%.
    • It was always 20% in mission, this was just a description error in the Arsenal stats!

Hotfix 33.0.5 (2023-05-03)

  • Fixed Melee Incarnon Weapons using the incorrect HUD indicator to state that Incarnon Mode was ready.

Hotfix 33.0.6 (2023-05-04)

  • Changed the following Skana variant’s “Elemental Excess” Incarnon Genesis evolution:
    • Prisma Skana: Increase Status Chance by 20% (was 30%)
    • Skana Prime: Increase Status Chance by 10% (was 24%)
      • Varying values for variants are intended for all Incarnon Genesis weapons, so the Skana was adjusted to match that format.

Update 33.0 (2023-04-26)

  • Introduced.
